:::info Файл `profiles.yml` предназначен только для dbt Core и dbt Fusion
Если вы используете <Constant name="dbt_platform" />, вам не нужно создавать файл `profiles.yml`. Этот файл требуется только в том случае, если вы используете dbt Core или dbt Fusion локально. Чтобы узнать больше о требованиях к Fusion, см. раздел [Supported features](/docs/fusion/supported-features). Чтобы подключить вашу платформу данных к <Constant name="cloud" />, см. раздел [About data platforms](/docs/cloud/connect-data-platform/about-connections).
:::
