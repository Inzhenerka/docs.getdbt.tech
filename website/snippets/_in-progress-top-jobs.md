## Мониторинг заданий {#job-monitoring}

На странице **Environments** есть два раздела, которые дают обзор заданий для выбранного окружения:

- **In progress** &mdash; Список заданий, которые в данный момент выполняются, с информацией о времени запуска
- **Top jobs by models built** &mdash; Ранжирует задания по количеству построенных моделей за определённый период времени
