- `adapter_response`: Словарь с метаданными, возвращаемыми базой данных, которые зависят от используемого адаптера. Например, это может быть `code` успешного выполнения, количество `rows_affected`, общее число `bytes_processed` и т. д. Не применяется для [data tests](/docs/build/data-tests).
    * `rows_affected` возвращает количество строк, изменённых последним выполненным выражением. В случаях, когда количество строк, затронутых запросом, невозможно определить или это неприменимо (например, при создании <Term id="view">view</Term>), для `rowcount` возвращается [стандартное значение](https://peps.python.org/pep-0249/#rowcount) `-1`.
