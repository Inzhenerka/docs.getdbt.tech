Чтобы отладить неудачный тест, найдите SQL, который выполнил dbt, следующим образом:

* <Constant name="cloud" />:
  * В выводе результатов теста нажмите на упавший тест, а затем выберите **"Details"**.

* <Constant name="core" />:
  * Откройте путь к файлу, который указан в сообщении об ошибке.
  * Перейдите в директорию `target/compiled/schema_tests`, где находятся все скомпилированные запросы тестов.

Скопируйте SQL в редактор запросов (в <Constant name="cloud" /> вы можете вставить его в новый `Statement`) и выполните запрос, чтобы найти записи, которые не прошли тест.
