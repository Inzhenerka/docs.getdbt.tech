---
По умолчанию dbt собирает модели в вашей целевой схеме. Чтобы изменить целевую схему:

* Если вы разрабатываете в **<Constant name="cloud" />**, эти настройки задаются для каждого пользователя при первом использовании среды разработки.
* Если вы разрабатываете с помощью **dbt Core**, это параметр `schema:` в вашем файле `profiles.yml`.

Если вы хотите распределить ваши модели по нескольким схемам, ознакомьтесь с документацией по [использованию пользовательских схем](/docs/build/custom-schemas).

Примечание: в BigQuery `dataset` используется как синоним `schema`.
