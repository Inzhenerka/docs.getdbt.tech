---
title: Когда следует запускать data tests?
description: "Запускать data tests в продакшене или при написании нового кода"
sidebar_label: "Когда запускать data tests"
id: when-to-test

---

Вам следует запускать data tests каждый раз, когда вы пишете новый код (чтобы убедиться, что изменения в SQL не сломали существующие модели), а также каждый раз, когда вы запускаете трансформации в production (чтобы убедиться, что ваши предположения о исходных данных по‑прежнему верны).
