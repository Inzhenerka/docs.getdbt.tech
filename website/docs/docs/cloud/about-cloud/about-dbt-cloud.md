<Constant name="dbt_platform" /> (dbt Cloud) — это самый быстрый и надёжный способ развертывания dbt. Разрабатывайте, тестируйте, планируйте выполнение, документируйте и исследуйте модели данных — всё в одном браузерном пользовательском интерфейсе.

Помимо предоставления размещённой архитектуры для запуска dbt в рамках всей вашей организации, <Constant name="cloud" /> оснащён готовыми инструментами для планирования заданий, CI/CD, хостинга документации, мониторинга и оповещений, интегрированной средой разработки (<Constant name="cloud_ide" />), а также позволяет разрабатывать и выполнять команды dbt из локального интерфейса командной строки (CLI) или редактора кода.

[Гибкие тарифные планы](https://www.getdbt.com/pricing/) и возможности <Constant name="cloud" /> делают его подходящим для команд по работе с данными любого размера — зарегистрируйтесь и воспользуйтесь [бесплатным 14‑дневным пробным периодом](https://www.getdbt.com/signup/)!
