### Многоарендный

Многоарендная (SaaS) среда развертывания относится к SaaS-приложению dbt Cloud, размещенному dbt Labs. Это наиболее часто используемое развертывание, которое полностью управляется и поддерживается dbt Labs, создателями dbt. Как продукт SaaS, пользователь может быстро [создать учетную запись](https://www.getdbt.com/signup/) на наших североамериканских серверах и сразу начать использовать dbt и связанные с ним сервисы. _Если вашей организации требуются облачные сервисы, размещенные в регионах EMEA или APAC_, пожалуйста, [свяжитесь с нами](https://www.getdbt.com/contact/). Развертывания размещаются на AWS или Azure ([Предварительный просмотр](/docs/dbt-versions/product-lifecycles#dbt-cloud)) и всегда обновляются до текущих поддерживаемых версий dbt, программных обновлений и исправлений ошибок.

### Одноарендный

Одноарендная среда развертывания предоставляет размещенную альтернативу многоарендной (SaaS) среде dbt Cloud. Хотя она все еще управляется и поддерживается dbt Labs, одноарендные экземпляры dbt Cloud предоставляют выделенную инфраструктуру в среде виртуального частного облака (VPC). Это достигается путем развертывания всей необходимой инфраструктуры с использованием повторно используемого развертывания Infrastructure as Code (IaC), построенного с помощью [Terraform](https://www.terraform.io/). Одноарендная инфраструктура размещается в выделенной учетной записи AWS или Azure и может быть настроена с определенными конфигурациями, такими как правила брандмауэра, для ограничения входящего трафика или размещения в определенных регионах.

Несколько распространенных причин выбора одноарендного развертывания вместо производственного SaaS-продукта включают:
- Требование, чтобы приложение dbt Cloud было размещено в выделенной VPC, которая логически отделена от инфраструктуры других клиентов
- Желание иметь несколько изолированных экземпляров dbt Cloud для тестирования, разработки и т.д.

_Чтобы узнать больше о настройке одноарендного развертывания dbt Cloud, [пожалуйста, свяжитесь с нашей командой продаж](mailto:sales@getdbt.com)._
