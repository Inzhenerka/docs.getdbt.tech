# Работа с dbt Copilot <Lifecycle status='managed, managed_plus'/> 

<IntroText>
Используйте <Constant name="copilot" />, чтобы создавать визуальные модели в <Constant name="visual_editor" /> с помощью запросов на естественном языке.

</IntroText>

[<Constant name="copilot" />](/docs/cloud/dbt-copilot) органично интегрируется с [<Constant name="visual_editor" />](/docs/cloud/canvas) — интерфейсом с поддержкой drag-and-drop, который помогает создавать визуальные модели с использованием запросов на естественном языке. Перед началом убедитесь, что у вас есть доступ к [<Constant name="visual_editor" />](/docs/cloud/use-canvas#access-canvas).
