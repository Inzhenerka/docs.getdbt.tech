Cloud IDE предоставляет горячие клавиши, функции и советы по разработке, которые помогут вам работать быстрее и быть более продуктивными.

<Constant name="cloud_ide" /> предоставляет сочетания клавиш, функции и советы по разработке, которые помогают работать быстрее и повышать продуктивность.

Используйте эту страницу <Constant name="cloud_ide" />, чтобы быстро находить справочную информацию по распространённым операциям.

| Описание сочетания клавиш | macOS | Windows |
|--------|----------------|------------------|
| Просмотреть полный список сочетаний клавиш редактора, которые помогают в разработке, например: добавление строчного комментария, изменение ширины табуляции, сборка изменённых моделей, изменение размера шрифта редактора и многое другое. | Fn-F1 | Fn-F1 |
| Выбрать файл для открытия. | Command-O | Control-O |
| Закрыть текущую активную вкладку редактора. | Option-W | Alt-W |
| Предпросмотр кода. | Command-Enter | Control-Enter |
| Скомпилировать код. | Command-Shift-Enter | Control-Shift-Enter |
| Показать список функций dbt в редакторе. | Ввести два символа подчёркивания `__` | Ввести два символа подчёркивания `__` |
| Открыть палитру команд для запуска команд и действий dbt. | Command-P / Command-Shift-P | Control-P / Control-Shift-P |
| Множественное редактирование в редакторе с помощью выделения нескольких строк. | Option-Click / Shift-Option-Command / Shift-Option-Click | Удерживать Alt и кликнуть |
| Открыть [**Invocation History Drawer**](/docs/cloud/studio-ide/ide-user-interface#invocation-history), расположенный в нижней части IDE. | Control-backtick (или Control + `) | Control-backtick (или Ctrl + `) | 
| Добавить блочный комментарий к выделенному коду. В SQL-файлах будет использоваться синтаксис Jinja `({# #})`, а не SQL-синтаксис `(/* */)`. | Shift-Option-A | Shift-Alt-A |
