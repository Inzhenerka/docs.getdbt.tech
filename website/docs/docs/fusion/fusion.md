<IntroText><Constant name="fusion_engine" /> — это движок следующего поколения, написанный на Rust, который лежит в основе разработки в рамках <Constant name="dbt_platform" /> (ранее dbt Cloud), а также локальной разработки в VS Code и Cursor и работы через CLI.</IntroText>

Fusion быстрее, умнее и экономичнее по затратам — он привносит понимание SQL и осведомлённость о состоянии проекта, мгновенную обратную связь и многое другое — во все рабочие процессы dbt, а также обеспечивает интегрированный опыт работы в VS Code через [расширение dbt](https://marketplace.visualstudio.com/items?itemName=dbtLabsInc.dbt) и [Language Server Protocol (LSP)](https://docs.getdbt.com/blog/dbt-fusion-engine-components#the-dbt-vs-code-extension-and-language-server). Это позволяет использовать такие возможности, как живые превью CTE, всплывающие подсказки, подсветку ошибок и многое другое.

Выберите один из следующих путей, чтобы начать работу с движком dbt Fusion.
