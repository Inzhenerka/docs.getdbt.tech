[Owners](/docs/build/groups) помогают определить пользователя или домен, ответственный за dbt-ассет. Для большинства ассетов владельцы задаются в коде вашего проекта с использованием groups. Исключением являются exposures: для downstream‑exposures, представляющих BI‑ассеты, владельцы автоматически подтягиваются из downstream‑инструмента на основе того, кому принадлежит этот ассет. Информацию о владельцах можно запрашивать через Discovery API.

В разделе [Example query](#example-query) показаны некоторые поля, которые можно запрашивать с помощью объекта `owners`. Полный список доступных полей смотрите в разделе [Fields](#fields), где представлена вся схема со всеми возможными полями для запроса.

### Пример запроса {#example-query}

Вы можете указать `environmentId` и значение `"exposure"` для `OwnerResourceType`, чтобы вернуть пользователей, владеющих exposures (downstream BI‑ассетами) в этом окружении, включая их контактную информацию.

```graphql
query {
  environment(id: 834) {
    applied {
      owners(resource: exposure) {
        email
        name
      }
    }
  }
}
```
