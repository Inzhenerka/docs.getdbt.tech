Объект **test** позволяет запрашивать информацию о конкретном тесте.

### Аргументы

При запросе `test` доступны следующие аргументы.

<NodeArgsTable parent="JobNode" name="test" />

Ниже приведены иллюстративные примеры запросов и описание схемы (всех возможных полей, которые можно запрашивать) объекта **test**.

### Пример запроса

Пример запроса ниже выводит информацию о тесте, включая состояние результата теста. В порядке убывания серьезности результат может принимать одно из следующих значений: `"error"`, `"fail"`, `"warn"` или `"pass"`.

```graphql
{
  job(id: 123) {
    test(uniqueId: "test.internal_analytics.not_null_metrics_id") {
      runId
      accountId
      projectId
      uniqueId
      name
      columnName
      state
    }
  }
}
```
