Объект `tests` позволяет запрашивать информацию обо всех тестах в рамках заданного job.

### Аргументы

При выполнении запроса к `tests` доступны следующие аргументы.

<NodeArgsTable parent="JobNode" name="tests" />

Ниже приведены иллюстративные примеры запросов, а также описание схемы (всех возможных полей, которые можно запрашивать) объекта `tests`.

### Пример запроса

Пример запроса ниже находит все тесты в данном job и включает информацию об этих тестах.

```graphql
{
  job(id: 123) {
    tests {
      runId
      accountId
      projectId
      uniqueId
      name
      columnName
      state
    }
  }
}
```
