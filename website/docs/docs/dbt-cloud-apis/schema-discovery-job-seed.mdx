Объект `seed` позволяет запрашивать информацию о конкретном seed в рамках заданного job.

### Аргументы

При выполнении запроса к `seed` доступны следующие аргументы.

<NodeArgsTable parent="JobNode" name="seed" />

Ниже приведены примеры запросов для иллюстрации, а также описание схемы объекта seed.

### Пример запроса

Пример ниже извлекает релевантную информацию о заданном seed. Например, вы можете посмотреть время загрузки.

```graphql
{
  job(id: 123) {
    seed(uniqueId: "seed.jaffle_shop.raw_customers") {
      database
      schema
      uniqueId
      name
      status
      error
    }
  }
}
```
