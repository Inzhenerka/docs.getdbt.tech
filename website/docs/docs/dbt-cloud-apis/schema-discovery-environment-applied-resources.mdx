Объект `resources` позволяет выполнять постраничный перебор всех ресурсов в вашем окружении.

В разделе [Пример запроса](#example-query) показаны некоторые поля, которые можно запрашивать с помощью объекта `resources`. Обратитесь к разделу [Fields](#fields), чтобы посмотреть полную схему, содержащую все возможные поля, доступные для запроса.

### Пример запроса {#example-query}

Вы можете указать `environmentId`, отфильтровать ресурсы по типу `"Model"` и ограничить выборку первыми 100 результатами, чтобы получить подробную информацию о первых 100 ресурсах-моделях в этом окружении, включая их метаданные, теги и расположение файлов:

```graphql
query {
  environment(id: 834) {
    applied {
      resources(
        filter: {
          types: [
            Model
          ]
        }, 
        first: 100
      ) {
        edges {
          node {
            accountId
            description
            environmentId
            filePath
            meta
            name
            projectId
            resourceType
            uniqueId
            tags
          }
        }
      }
    }
  }
}
```
