[dbt packages](/docs/build/packages) — это библиотеки с моделями, макросами и другими ресурсами, которые решают определённую область задач и используются в проектах dbt. Вы можете запрашивать пакеты проекта через Discovery API.

В разделе [Example query](#example-query) показаны некоторые поля, которые можно запрашивать с помощью объекта `packages`. Обратитесь к разделу [Fields](#fields), чтобы просмотреть полную схему, содержащую все возможные поля, доступные для запроса.

### Аргументы {#arguments}

При выполнении запроса к `packages` вы можете использовать следующие аргументы:

<NodeArgsTable parent="AppliedState" name="packages" />

### Пример запроса {#example-query}

Вы можете указать `environmentId` и `"model"` в качестве типа ресурса, чтобы увидеть все dbt‑пакеты в этом окружении, которые содержат ресурсы моделей:

```graphql
query {
  environment(id: 834) {
    applied {
      packages(resource: "model")
    }
  }
}
```
