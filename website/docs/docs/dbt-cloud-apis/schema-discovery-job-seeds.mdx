Объект `seeds` позволяет запрашивать информацию обо всех seed-файлах в рамках заданного job.

### Аргументы {#arguments}

При выполнении запроса к `seeds` доступны следующие аргументы.

<NodeArgsTable parent="JobNode" name="seed" />

Ниже приведены иллюстративные примеры запросов, а также описание схемы объекта `seeds`.

### Пример запроса {#example-query}

В примере ниже показан запрос, который извлекает релевантную информацию обо всех seed-файлах в заданном job. Например, так можно посмотреть время загрузки.

```graphql
{
  job(id: 123) {
    seeds {
      uniqueId
      name
      executionTime
      status
    }
  }
}
```
