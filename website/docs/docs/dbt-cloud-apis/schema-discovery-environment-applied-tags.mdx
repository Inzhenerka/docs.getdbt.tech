[Tags](/reference/resource-configs/tags) предоставляют механизм для категоризации и группировки ресурсов внутри проекта dbt, что позволяет избирательно выполнять и управлять этими ресурсами. Вы можете запрашивать теги через Discovery API.

В разделе [Example query](#example-query) показаны несколько полей, которые можно запрашивать с помощью объекта `tags`. Обратитесь к разделу [Fields](#fields), чтобы посмотреть полную схему, содержащую все возможные поля, доступные для запроса.

### Example query

Вы можете использовать `environmentId`, чтобы получить имена всех тегов в вашем окружении:

```graphql
query {
  environment(id: 834) {
    applied {
      tags {
        name
      }
    }
  }
}
```
