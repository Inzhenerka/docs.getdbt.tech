[Тайлы состояния экспозиции](/docs/explore/data-tile) сводят сигналы о состоянии данных для потребителей данных и могут быть встроены в downstream‑инструменты. Информацию об этих тайлах можно запрашивать через Discovery API.

В разделе [Пример запроса](#example-query) показаны некоторые поля, которые можно запрашивать с помощью объекта `exposureTile`. Полный список доступных полей см. в разделе [Поля](#fields), где приведена вся схема со всеми возможными полями для запроса.

### Пример запроса

Вы можете указать `environmentId` и отфильтровать по `uniqueId` модели, чтобы получить информацию о качестве данных и метаданных для тайла экспозиции, связанного с моделью `customers` в пакете `marketing`:

```graphql
query {
  environment(id: 834) {
    applied {
      exposureTile(
        filter: {uniqueId: "model.marketing.customers"} # Use this format for unique ID: RESOURCE_TYPE.PACKAGE_NAME.RESOURCE_NAME
      ) {
        accountId # The account ID of this node
        environmentId
        projectId
        exposureType
        filePath
        quality
      }
    }
  }
}
```
