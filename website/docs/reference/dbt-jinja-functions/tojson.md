Метод контекста `tojson` может быть использован для сериализации примитивного объекта Python, например, `dict` или `list`, в строку JSON.

__Args__:
 * `value`: Значение, которое нужно сериализовать в JSON (обязательный параметр)
 * `default`: Значение по умолчанию, которое будет возвращено, если аргумент `value` не может быть сериализован (необязательный параметр)

### Использование: {#usage}
```
{% set my_dict = {"abc": 123} %}
{% set my_json_string = tojson(my_dict) %}

{% do log(my_json_string) %}
```
