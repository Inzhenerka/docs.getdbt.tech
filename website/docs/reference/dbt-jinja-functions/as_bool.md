Jinja-фильтр `as_bool` приводит результат, скомпилированный Jinja, к булевому
значению (`True` или `False`) либо возвращает ошибку, если его невозможно
представить в виде булевого значения.

### Использование: {#usage}

В примере ниже фильтр `as_bool` используется для приведения Jinja-выражения,
чтобы включить или отключить набор моделей в зависимости от значения `target`.

<File name='dbt_project.yml'>

```yml
models:
  my_project:
    for_export:
      enabled: "{{ (target.name == 'prod') | as_bool }}"
```

</File>
